# Add-AzureAccounts
## SYNOPSIS
Wraps Add-AzureAccount and Add-AzureRmAccount.

## SYNTAX

```
Add-AzureAccounts
```

## DESCRIPTION
Logs you in ASM and ARM by wrapping Add-AzureAccount and Add-AzureRmAccount.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```

```

This command will log you in both ASM and ARM.
